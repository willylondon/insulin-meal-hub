Option 1: Replace Entire "21 Recipes" Sheet (Recommended)

Use this to get all 30 meals at once. Your existing 5 meals will be replaced.

  1. In Google Sheets, click the "21 Recipes" tab
  2. Click the arrow next to the tab name → "Delete"
  3. Click "File" → "Import"
  4. Upload 30-meals-database.csv
  5. Select "Replace current sheet"
  6. Choose "21 Recipes" as the sheet name
  7. Click "Import data"
  8. The sheet should now have 31 rows (header + 30 meals)

Option 2: Append 25 New Meals

If you want to keep your existing 5 meals and add 25 new ones.

  1. Open 25-new-meals.csv in Excel/Sheets
  2. Select all 25 data rows (excluding header)
  3. Copy (Ctrl+C)
  4. Go to your Google Sheet → "21 Recipes" tab
  5. Scroll to the bottom (after row 6)
  6. Select cell A7
  7. Paste (Ctrl+V)
  8. The sheet should now have 31 rows (header + 30 meals)

🚀 Deploy to Vercel with GitHub

  1. Create a new GitHub repository (e.g., insulin-meal-hub)
  2. Upload all files from this folder to the repository
  3. Go to vercel.com and sign in with GitHub
  4. Click "Add New..." → "Project"
  5. Select your repository
  6. Click "Deploy" (no configuration needed)
  7. Your app will be live at https://insulin-meal-hub.vercel.app

Vercel automatically detects static sites and serves them for free.
